# Cheatsheet
Personal cheatsheet for Helix, LaTeX and Markdown

## Helix

### basic 
- k &#8594; up
- j &#8594; down
- h &#8594; left
- l &#8594; right
- i &#8594; input mode
- v &#8594; visual mode
- esc &#8594; command mode

#### delete / replace
- d &#8594; delete
- r &#8594; replace 
- R &#8594; replaced with copy

#### command and visual mode
- u &#8594; undo
- U &#8594; redo
- w &#8594; select word to the end
- b &#8594; select word to the start
- x &#8594; select line
- o &#8594; new line below
- O &#8594; new line above
- c &#8594; copy
- p &#8594; past

##### go to
- g 2 g &#8594; line 2, works with every line number
- g l &#8594; line end
- g h &#8594; line start
